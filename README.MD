# Quiz Program

Многопоточная клиент-серверная система на Java.
Несколько клиентов параллельно генерируют запросы на прохождение теста, сервер складывает их в общую очередь, а потоки конкурентно обрабатывают запросы и сохраняют результаты.

## Описание программы

Система реализует модель producer-consumer:

- **Клиенты (`QuizClientGenerator`)** создают `QuizRequest` и отправляют на сервер.
- **Сервер (`QuizServer`)** принимает запросы и помещает их в `BlockingQueue`.
- **Исполнители (`QuizRequestProcessor`)** забирают задачи из очереди, загружают тест через `FileTestLoader/PlainTestParser` и выполняют обработку.
- **Хранилище результатов (`QuizResultStore`)** потокобезопасно сохраняет результаты.

Корректное завершение работы реализовано через **poison pill**: после окончания генерации запросов в очередь отправляются запросы остановки, и каждый поток завершает работу без принудительного прерывания.

### Потокобезопасность

- Очередь запросов защищена `LinkedBlockingQueue`.
- Доступ к общему хранилищу результатов защищен `ReentrantLock` (Mutex).
- Уникальные ID запросов формируются через `AtomicInteger`.

## Сборка и запуск

```bash
git clone -b concurrency https://github.com/cyberc3dr/quizprogram
cd quizprogram
gradle clean build
java -jar build/quizprogram.jar
```
