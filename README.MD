# quizprogram

1. Clone the repository
```
git clone https://github.com/cyberc3dr/quizprogram
git checkout spring
```
OR
```
git clone -b spring https://github.com/cyberc3dr/quizprogram
```

2. Open the project in your IDE.
3. Build
```
gradle clean build
```
4. Run
```
java -jar build/quizprogram-all.jar
```

5. Test file is `test.txt`
